# WINDOWS-SERVER
#small office network windows server
small office network design and implementation using windows server with domain controller and DNS configuration 

## project overview 
This project demonstrates the implementation of a small enterprise network using window server as domain controller and DNS server 

## Network Architecture 
-1 Domain controller (windows server)
-2 client Machines 
- LAN Network using virtualbox

## Technologies used
- TCP/IP
- DNS
- Active Directory
- Windows Server
- Virtualbox

## IP Configuration
Server: 192.168.1.10
Client1: 192.168.1.11
Client2: 192.168.1.12

## Testing Commands
-ping
-ipconfig
-nslookup

## Outcome
Succesfull domain join, DNS resolution and communication between clients and server
